

**一、实验目的**

掌握Logisim中通过使用真值表、表达式等方式设计并实现数码管输出。

**二、实验内容**

对于4位二进制数，可以表示出0~15这16个数，在八段共阳极数码管上则依次显示0，1，2，……，9，A，B，C，d，E，F（见图1）。要求使用Logisim画出上述对应的电路。（具体实验内容见实验视频指导）
